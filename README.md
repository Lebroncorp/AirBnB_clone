# AirBnB clone - The Console.

#### How To Use:
##### To launch:
./console.py

##### To exit:
$ quit
or
$ <Ctrl + D>

##### To create:
$ create <class_name>

##### To display all:
$ all
or
$ all <class_name>

##### To display a specific data:
$ show <class_name> <id>

##### To update or modify:
$ update <class_name> <id> <attribute_name> <attribute_value>

##### To delete:
$ destroy <class_name> <id>
